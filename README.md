# Power BI Audiobook Analysis Project

##  Project Overview
This project focuses on cleaning audiobook data using Excel and visualizing insights using Power BI.

##  Data Cleaning (Excel)
- Removed duplicates
- Handled null values
- Standardized text formatting
- Cleaned rating and review columns

##  Power BI Visualizations
- Top authors by number of audiobooks
- Rating distribution
- Price range analysis
- Review vs rating insights
- Category-wise audiobook analysis

##  Tools Used
- Microsoft Excel
- Power BI
- GitHub
